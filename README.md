# Gestor de Contactos

Aplicación de consola en Java para gestionar contactos.

## Funcionalidades
- Agregar contacto
- Ver todos los contactos
- Buscar contacto por nombre
- Eliminar contacto

## Tecnologías
- Java
- POO

## Cómo correrlo
1. Clonar el repositorio
2. Abrir en NetBeans
3. Ejecutar GestorContactos.java
